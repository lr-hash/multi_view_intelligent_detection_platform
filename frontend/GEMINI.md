# Frontend Module

This module contains the frontend application, built with Vue.js 3.

## Responsibilities

-   **User Interface**: Provides an industrial-grade "command center" UI for real-time monitoring and management.
-   **Data Visualization**: Implements 3D scene rendering (Three.js) for borehole trajectories and 2D analytics (Chart.js) for trends.
-   **Dashboards**: Displays real-time data fusion results and alarm statuses.
-   **User Interaction**: Manages interactive 3D model manipulation and strict workflow-driven administrative tasks.
-   **State Management**: Manages application-wide monitoring state and authentication using Pinia.
