# Frontend Module

This module contains the frontend application, built with Vue.js.

## Responsibilities

-   **User Interface**: Provides a web-based user interface for interacting with the platform.
-   **Data Visualization**: Implements 2D and 3D visualizations for fracturing design, drilling trajectories, and monitoring data.
-   **Dashboards**: Displays real-time and historical data through interactive charts and dashboards.
-   **User Interaction**: Handles user input, queries, and interactions with the backend API.
-   **State Management**: Manages the application's state using Pinia.
