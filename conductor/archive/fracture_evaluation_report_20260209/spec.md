# Specification - 压裂效果评价模型与 PDF 报告生成

## 1. 概述
本轨道旨在实现平台的核心分析价值：压裂效果评价。通过综合分析多源监测数据，量化评估压裂工程对矿压控制的实际效果，并生成标准化的 PDF 报告。

## 2. 需求分析
### 2.1 评价模型研发
- **指标对比**：对比压裂区域与未压裂区域（或压裂前后）的平均支架压力、微震总能量、变形速率。
- **有效率计算**：基于预设算法计算压裂有效率（0-100%）。
- **顶板稳定性评级**：给出最终的稳定性等级建议。

### 2.2 报告生成逻辑
- **内容结构**：包含基本信息、施工概况、多源指标对比图、评价结论、防治建议。
- **PDF 渲染**：后端将分析结果与图表渲染为 PDF 文件。
- **异步处理**：由于报告生成较慢，应支持异步任务。

### 2.3 前端展示
- **报告列表**：展示已生成的报告。
- **在线预览**：支持 PDF 在线预览与下载。

## 3. 验收标准
- [ ] 后端实现评价算法，并通过逻辑验证。
- [ ] 成功生成包含图表和文字描述的 PDF 报告文件。
- [ ] 提供 `/api/v1/evaluation/report/<borehole_id>` 获取报告详情/文件的接口。
- [ ] 前端能够展示评价结果并提供下载链接。
