# Specification - 压裂施工 3D 可视化与动态效果实现

## 1. 概述
本轨道旨在 3D 场景中增强压裂施工的表现力。通过将抽象的压力、流量数值转化为具象的 3D 实体（如压裂球体），帮助技术人员直观判断压裂作业的影响范围和质量。

## 2. 需求分析
### 2.1 后端数据支持
- **数据查询接口**：提供按钻孔获取压裂施工实时/历史数据的 API。
- **数据关联**：确保压裂段号与钻孔轨迹中的特定位置（测深）能够对应。

### 2.2 3D 渲染增强
- **压裂球渲染**：在钻孔轨迹的压裂段位置渲染半透明球体。
- **动态映射逻辑**：
    - **大小（Radius）**：球体半径与“施工压力 (Pressure)”正相关。
    - **颜色（Color）**：颜色从绿到红映射“累计排量 (Total Volume)”或“瞬时流量 (Flow Rate)”。
- **信息交互**：点击压裂球可展示该段的详细施工参数。

## 3. 验收标准
- [ ] 后端提供 `/api/v1/visualization/fracture-data/<borehole_id>` 接口。
- [ ] 3D 场景中能够在钻孔轨迹上正确显示压裂段球体。
- [ ] 球体的大小和颜色随模拟数据的变化而变化。
- [ ] 点击球体能够触发 UI 面板显示详细数据。
