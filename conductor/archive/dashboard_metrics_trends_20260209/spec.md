# Specification - 数据看板核心指标与趋势分析展示

## 1. 概述
本轨道旨在构建平台的核心监控页面——数据看板。通过 2D 图表和关键数值卡片，为用户提供直观的工作面安全状态概览，并支持对历史数据的深度分析。

## 2. 需求分析
### 2.1 核心指标卡片
- **矿压 (Strata Pressure)**：展示当前平均支架压力。
- **微震 (Microseismic)**：展示最近一次事件能量及今日事件总数。
- **变形 (Deformation)**：展示测点的累计位移量。
- **压裂 (Fracturing)**：展示当前施工段号及施工状态。

### 2.2 趋势分析图表 (Chart.js)
- **矿压趋势图**：时序展示压力波动曲线。
- **微震频次图**：柱状图展示不同能量等级的事件分布或小时频次。
- **位移速率图**：展示巷道变形的演化速率。

### 2.3 交互需求
- **时间范围切换**：支持 24h, 7d, 30d 等维度的视图切换。
- **实时刷新**：核心指标应根据 WebSocket 或定时轮询进行更新。

## 3. 验收标准
- [ ] DashboardView 页面布局符合“工业专业化”准则。
- [ ] 成功集成 Chart.js 且能够正确渲染各维度趋势图。
- [ ] 能够通过 API 获取并展示后端提供的模拟/真实指标数据。
- [ ] 切换时间维度时，图表数据能够正确响应更新。
