# Gemini Project: 多视域智能侦测平台

This project is a full-stack web application designed to provide an intelligent monitoring and analysis platform for mine safety. It integrates data from multiple sources, provides visualization capabilities, and offers intelligent evaluation of engineering effects.

## Modules

- **backend**: A Flask-based backend application responsible for data processing, analysis, and serving the API.
- **frontend**: A Vue.js-based frontend application for user interaction, data visualization, and dashboarding.
